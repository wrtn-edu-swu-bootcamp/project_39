# 그룹 챌린지 어플리케이션 서비스 기획안

## 1. 서비스 개요

### 1.1 문제 정의

현재 많은 청년들이 겪고 있는 세 가지 핵심 문제:

- **은둔고립청년**: 사회적 고립 상태에 있는 청년들
- **지지체계 부족**: 일상적인 활동을 함께 할 동료나 지지 네트워크의 부재
- **성취감 부족**: 작은 성취를 인정받고 공유할 수 있는 기회의 부족

### 1.2 해결 방안

그룹 기반 루틴 챌린지를 통해 사용자들이 함께 목표를 달성하고, 일상적인 성취를 공유하며, 지지 네트워크를 형성할 수 있는 플랫폼을 제공합니다.

### 1.3 타겟 사용자

- **주요 타겟**: 청년층 전반 (20-30대)
- **핵심 사용자**: 은둔고립청년, 일상 루틴을 지속하기 어려운 청년, 성취감을 느끼고 싶은 청년

### 1.4 핵심 가치 제안

- **함께하는 성취**: 혼자가 아닌 그룹과 함께하는 루틴 수행으로 지속성 확보
- **시각적 성취 기록**: "지나온 길"을 통해 자신의 여정을 되돌아보고 성취감 획득
- **자유로운 커뮤니티**: 취미공유방을 통한 자연스러운 소통과 연결
- **공정한 참여 환경**: 신고 시스템을 통한 건전한 챌린지 문화 유지

## 2. 주요 기능

### 2.1 그룹 시스템

#### 2.1.1 그룹 생성 및 참여
- **그룹 생성 권한**: 모든 사용자가 그룹을 생성할 수 있음
- **그룹 주제**: 사용자가 자유롭게 주제를 설정 (운동, 독서, 공부, 취미 등 제한 없음)
- **그룹 기간**: 그룹장이 3주부터 3개월까지 원하는 기간을 설정
- **그룹 인원**: 그룹장이 최대 인원수를 자유롭게 설정 가능

#### 2.1.2 그룹 구조
- **그룹장**: 그룹 생성자, 그룹 기간 및 인원 제한 설정 권한 보유
- **그룹 멤버**: 그룹에 참여하여 루틴 인증을 수행하는 사용자들

#### 2.1.3 그룹장 권한 및 책임
- **그룹 설정 관리**: 그룹명, 주제, 기간, 최대 인원수 변경 (단, 그룹 시작 후에는 기간 변경 불가)
- **멤버 관리**: 
  - 그룹 참여 신청 승인/거절 (수동 승인 모드인 경우)
  - 멤버 강제 퇴출 권한 (부적절한 행위 시)
- **그룹 해체**: 그룹 시작 전에만 그룹 해체 가능
- **그룹 비활성화**: 그룹 기간 종료 후 자동으로 비활성화되며, 기록은 보관됨

#### 2.1.4 그룹 참여 프로세스
- **참여 신청**: 사용자가 관심 있는 그룹에 참여 신청
- **승인 방식**: 
  - **자동 승인**: 그룹장이 설정한 경우 즉시 참여 가능
  - **수동 승인**: 그룹장이 참여 신청을 검토 후 승인/거절
- **인원 제한**: 최대 인원수에 도달한 경우 대기열 기능 제공 (선착순 또는 그룹장 선택)
- **참여 제한**: 동일 그룹에 중복 참여 불가

#### 2.1.5 그룹 탈퇴 및 재참여
- **그룹 탈퇴**: 그룹 기간 중 언제든지 탈퇴 가능 (단, 탈퇴 시 해당 그룹의 All Clear 달성 불가)
- **탈퇴 후 기록**: 탈퇴 전까지의 인증 기록은 "지나온 길"에 보관됨
- **재참여**: 탈퇴한 그룹에 재참여 가능 (단, 그룹 기간이 진행 중이고 인원이 남아있는 경우)
- **그룹 종료 후**: 종료된 그룹은 재참여 불가, 기록만 조회 가능

### 2.2 루틴 인증 시스템

#### 2.2.1 인증 방식
- **인증 형식**: 사진을 통한 루틴 수행 인증
- **게시 형식**: 패들렛(Padlet) 형식으로 게시물을 올림
  - **UI/UX**: 카드 형태의 게시물을 그리드 레이아웃으로 배치
  - **표시 방식**: 시간순 또는 최신순으로 정렬 가능
  - **게시물 구성**: 사진, 작성자, 작성 시간, 댓글 수 표시
  - **인터랙션**: 카드 클릭 시 상세 보기, 댓글 작성 가능
- **인증 빈도**: 매일 각자 정해진 루틴을 인증

#### 2.2.2 소통 기능
- **댓글 반응**: 루틴 인증 게시물에 댓글로 반응 가능
- **채팅 기능**: 그룹 내 채팅은 불가 (게시물과 댓글을 통한 소통만 가능)

### 2.3 신고 시스템

#### 2.3.1 신고 대상
- 무관한 루틴 인증: 그룹 주제와 관련 없는 인증 사진
- 유해한 사진: 부적절하거나 유해한 내용의 사진

#### 2.3.2 신고 처리
- **처리 방식**: 
  - **1차 신고**: 관리자 검토 후 경고 조치
  - **2차 신고 (누적 2건)**: 해당 게시물 자동 삭제 및 경고
  - **3차 신고 (누적 3건)**: 그룹 내 활동 3일 정지
  - **4차 신고 이상 (누적 4건 이상)**: 그룹 내 활동 7일 정지 또는 영구 정지
- **신고 검토**: 관리자가 신고 내용을 검토하여 부당한 신고는 무효 처리
- **목적**: 건전한 챌린지 환경 유지 및 무관한 인증 방지

#### 2.3.3 이의 제기 절차
- **이의 신청**: 제재 받은 사용자는 이의 신청 가능
- **재검토**: 관리자가 신고 내용과 이의 신청을 재검토
- **결과 통보**: 재검토 결과를 사용자에게 통보

### 2.4 보상 시스템

#### 2.4.1 일일 인증 보상
- **보상 형태**: 
  - **포인트**: 루틴 인증 완료 시 포인트 획득 (예: 10포인트)
  - **경험치(EXP)**: 사용자 레벨 상승을 위한 경험치 획득
  - **배지**: 특정 조건 달성 시 배지 획득 (예: 연속 인증 배지)
- **보상 수령**: 인증 완료 즉시 자동 지급
- **목적**: 일일 성취감 제공 및 지속적 참여 동기 부여

#### 2.4.2 All Clear 보상
- **All Clear 기준**: 그룹 기간 동안 모든 루틴 인증을 완료한 경우
- **보상 형태**: 
  - **특별 포인트**: 일반 인증 보상의 3배 포인트 (예: 30포인트)
  - **특별 배지**: All Clear 달성 배지 (그룹별 또는 통합)
  - **골드 테두리**: "지나온 길"에서 All Clear 달성 그룹에 골드 테두리 표시
  - **성취 인증서**: All Clear 달성 인증서 (이미지 형태로 저장 가능)
- **표시 방식**: "지나온 길"에서 All Clear 달성 그룹은 더 눈에 띄게 표시 (골드 테두리, 반짝임 효과)

#### 2.4.3 참여도별 차등 표시
- **All Clear (100%)**: 가장 눈에 띄게 표시 (더 빤짝이게)
- **일반 참여 (20% 이상)**: 평범하게 표시
- **낮은 참여 (20% 미만)**: 낮은 채도 색상(어두운 회색 톤)으로 표시하여 낮은 참여도임을 시각적으로 표현

### 2.5 취미공유방

#### 2.5.1 기능 개요
- 그룹 참여와 무관한 메인 화면의 커뮤니티 공간
- 우울감 타파를 위한 취미 공유 및 모집 공간

#### 2.5.2 게시 규칙
- **작성 제한**: 하루 최대 1개 글만 작성 가능 (도배 방지)
- **게시 내용 예시**:
  - "쿠키런 같이 하실분! ID = 블라블라"
  - "우울할 때 듣는 노래 추천드려요"
  - 취미 관련 정보 공유 및 함께할 사람 모집

### 2.6 식물키우기

#### 2.6.1 기능 개요
- 루틴 인증과 연계된 시각적 성취 표현 기능
- 사용자의 루틴 인증 활동에 따라 가상 식물이 성장하는 시스템

#### 2.6.2 작동 방식
- **식물 선택**: 사용자가 원하는 식물 종류 선택 (초기 제공 또는 포인트로 구매)
- **성장 조건**: 
  - 일일 인증 완료 시 식물에 물주기 (성장도 증가)
  - 연속 인증 시 추가 성장 보너스
  - All Clear 달성 시 특별 꽃 피우기 또는 열매 수확
- **시각적 표현**: 식물의 성장 단계를 애니메이션으로 표현 (새싹 → 잎 → 꽃 → 열매)
- **수집 요소**: 다양한 식물 종류 수집 가능 (배지 시스템과 연계)

#### 2.6.3 목적
- 루틴 인증의 시각적 동기 부여
- 성취감을 시각적으로 표현하여 지속적 참여 유도
- 게이미피케이션 요소를 통한 재미 요소 추가

### 2.7 알림 시스템

#### 2.7.1 알림 종류
- **루틴 인증 리마인더**: 매일 설정한 시간에 루틴 인증 알림
- **그룹 활동 알림**: 
  - 새로운 멤버 참여 알림
  - 다른 멤버의 인증 게시물 업로드 알림
  - 댓글 및 반응 알림
- **그룹 기간 종료 알림**: 그룹 기간 종료 전 3일, 1일 전 알림
- **보상 수령 알림**: 일일 보상 및 All Clear 보상 수령 알림
- **신고 처리 알림**: 신고 접수 및 제재 조치 알림

#### 2.7.2 알림 설정
- 사용자가 알림 종류별로 ON/OFF 설정 가능
- 알림 시간대 설정 가능
- 무음 시간대 설정 가능

### 2.8 그룹 검색 및 추천

#### 2.8.1 그룹 검색
- **키워드 검색**: 그룹명, 주제, 설명으로 검색
- **필터링 기능**:
  - 그룹 기간별 필터 (3주, 1개월, 2개월, 3개월)
  - 주제별 필터 (운동, 독서, 공부, 취미 등)
  - 인원 수 필터 (빈 자리 있는 그룹만 보기)
  - 참여 방식 필터 (자동 승인, 수동 승인)
- **정렬 기능**: 최신순, 인기순, 기간순 정렬

#### 2.8.2 그룹 추천
- **개인화 추천**: 사용자의 관심사 및 참여 이력 기반 추천
- **인기 그룹**: 현재 활발한 그룹 추천
- **새로운 그룹**: 최근 생성된 그룹 추천
- **비슷한 관심사**: 사용자가 참여한 그룹과 유사한 주제의 그룹 추천

### 2.9 사용자 프로필

#### 2.9.1 프로필 구성
- **기본 정보**: 닉네임, 프로필 사진, 자기소개
- **활동 통계**: 
  - 참여한 그룹 수
  - All Clear 달성 횟수
  - 총 인증 횟수
  - 현재 참여 중인 그룹 수
- **배지 컬렉션**: 획득한 배지 목록
- **레벨 시스템**: 루틴 인증 활동에 따른 레벨 표시

#### 2.9.2 프로필 커스터마이징
- 프로필 테마 변경 (포인트로 구매 가능)
- 배지 배치 및 표시 설정
- 공개/비공개 설정

### 2.10 지나온 길

#### 2.10.1 기능 개요
개인의 그룹 참여 여정을 시각적으로 기록하고 확인할 수 있는 기능

#### 2.10.2 화면 구성

**첫 페이지: "00님의 여정"**
- 사용자 개인 여정의 시작점
- 개인화된 여정 소개

**2페이지부터: 지나간 사람들**
- "00님과 0번째 만남" 형식으로 표시
- **만남 기준**: 같은 그룹에 참여했던 사람들 중에서
  - 댓글을 주고받은 사람 우선 표시
  - 같은 그룹에서 함께 활동한 사람 표시
  - 만남 횟수는 같은 그룹에 참여한 횟수로 계산
- 과거에 만났던 사람들의 기록 및 함께한 그룹 정보 표시

**그 다음: 그룹 참여 여정**
- 그룹 참여 이력이 시간순으로 표시
- 각 그룹별 정보:
  - **기간**: 00.00.00 ~ 00.00.00 형식
  - **그룹명**: 참여한 그룹 이름
  - **참여도 표시**:
    - All Clear: 더 빤짝이게 표시
    - 일반 참여 (20% 이상): 평범하게 표시
    - 낮은 참여 (20% 미만): 낮은 채도 색상(어두운 회색 톤)으로 표시

#### 2.10.3 시각화 목적
- 자신의 성장 과정을 되돌아보며 성취감 획득
- 지속적인 참여 동기 부여
- 개인 여정의 시각적 기록

## 3. 사용자 여정

### 3.1 신규 사용자 온보딩
1. 앱 설치 및 회원가입 (소셜 로그인 또는 이메일 회원가입)
2. 서비스 소개 및 주요 기능 안내 (온보딩 튜토리얼)
3. 첫 그룹 탐색 또는 생성

### 3.2 그룹 탐색 및 참여
1. 관심 있는 주제의 그룹 탐색 (검색, 필터링, 추천 기능 활용)
2. 그룹 정보 확인 (기간, 인원, 주제, 참여 방식 등)
3. 그룹 참여 신청
4. 그룹장 승인 또는 자동 참여 (그룹 설정에 따라)
5. 참여 승인 후 그룹 활동 시작
6. 알림 설정 (루틴 인증 리마인더 등)

### 3.3 루틴 인증 프로세스
1. 매일 정해진 시간에 루틴 수행
2. 루틴 수행 사진 촬영
3. 패들렛 형식으로 인증 게시물 작성 및 업로드
4. 다른 멤버들의 인증 확인 및 댓글 반응
5. 일일 인증 보상 수령

### 3.4 성취 확인 및 보상 수령
1. 일일 인증 보상 확인
2. "지나온 길"에서 개인 여정 확인
3. 그룹 기간 종료 시 참여도 확인
4. All Clear 달성 시 특별 보상 수령

### 3.5 취미공유방 활용
1. 메인 화면에서 취미공유방 접근
2. 다른 사용자들의 취미 공유 글 확인
3. 하루 1개 글 작성 제한 내에서 취미 공유 또는 모집 글 작성
4. 댓글을 통한 소통

## 4. 기능 상세

### 4.1 그룹 생성 프로세스

1. **그룹 생성 버튼 클릭**
2. **그룹 정보 입력**:
   - 그룹명
   - 그룹 주제/설명
   - 그룹 기간 선택 (3주 ~ 3개월)
   - 최대 인원수 설정
3. **그룹 생성 완료**
4. **그룹 공개 및 멤버 모집 시작**

### 4.2 루틴 인증 규칙

- **인증 주기**: 매일 1회 (그룹장이 설정한 루틴 기준)
- **인증 방법**: 사진 업로드 필수
- **인증 시간**: 사용자가 자유롭게 선택 (그룹별로 권장 시간 설정 가능)
- **인증 실패**: 해당 일 인증 누락 (All Clear 달성 불가)

### 4.3 신고 시스템 작동 방식

1. **신고 버튼 클릭**: 무관한 인증 또는 유해 사진 신고
2. **신고 사유 선택**: 신고 사유 선택 및 상세 내용 입력
   - 무관한 루틴 인증
   - 유해한 사진
   - 스팸 또는 도배
   - 기타 부적절한 행위
3. **신고 처리**: 
   - 신고 접수 후 관리자 검토
   - 신고 누적 시 단계별 제재 시스템 작동
4. **제재 수준**:
   - **1차 (1건)**: 경고
   - **2차 (2건)**: 게시물 삭제 및 경고
   - **3차 (3건)**: 그룹 내 활동 3일 정지
   - **4차 이상 (4건 이상)**: 그룹 내 활동 7일 정지 또는 영구 정지
5. **이의 제기**: 제재 받은 사용자는 이의 신청 가능 (관리자 재검토)

### 4.4 참여도 계산 방식

- **참여도 (%) = (실제 인증 횟수 / 전체 인증 가능 횟수) × 100**
- **전체 인증 가능 횟수**: 그룹 기간 동안의 총 일수
- **실제 인증 횟수**: 사용자가 실제로 인증한 횟수

### 4.5 All Clear 기준

- **All Clear 조건**: 그룹 기간 동안 모든 날짜의 루틴 인증을 완료
- **인증률**: 100% (모든 날짜 인증 완료)
- **보상**: 특별 보상 제공 및 "지나온 길"에서 특별 표시

### 4.6 기술적 세부사항

#### 4.6.1 사진 업로드 제한
- **파일 형식**: JPG, JPEG, PNG 형식 지원
- **파일 크기**: 최대 10MB
- **사진 개수**: 인증당 1장 (필수)
- **이미지 처리**: 자동 리사이징 및 압축 (서버 부하 최소화)

#### 4.6.2 사용자 인증 방식
- **소셜 로그인**: 카카오, 구글, 네이버 소셜 로그인 지원
- **이메일 회원가입**: 이메일 및 비밀번호 기반 회원가입 지원
- **계정 연동**: 여러 소셜 계정 연동 가능
- **로그인 유지**: 자동 로그인 기능 제공

#### 4.6.3 플랫폼 정보
- **모바일 앱**: iOS, Android 네이티브 앱 제공
- **웹 버전**: 반응형 웹 지원 (향후 확장 검토)
- **최소 지원 버전**: 
  - iOS 13.0 이상
  - Android 8.0 (API 26) 이상

#### 4.6.4 오프라인 지원
- **오프라인 모드**: 제한적 지원
  - 오프라인 상태에서도 이전에 로드된 그룹 정보 및 게시물 조회 가능
  - 오프라인 상태에서는 인증 업로드 불가 (네트워크 연결 필요)
- **동기화**: 네트워크 연결 시 자동 동기화

## 5. 참여도 및 보상 체계

### 5.1 참여도 구분

| 참여도 | 기준 | 표시 방식 | 설명 |
|--------|------|-----------|------|
| All Clear | 100% | 더 빤짝이게 표시 | 모든 루틴 인증 완료 |
| 일반 참여 | 20% 이상 | 평범하게 표시 | 일정 수준 이상 참여 |
| 낮은 참여 | 20% 미만 | 낮은 채도 색상(어두운 회색 톤)으로 표시 | 참여도가 낮음을 시각적으로 표현 |

### 5.2 보상 종류 및 조건

#### 5.2.1 일일 인증 보상
- **조건**: 매일 루틴 인증 완료
- **보상 내용**: 
  - 포인트 10점
  - 경험치(EXP) 획득
  - 연속 인증 시 추가 배지 (3일, 7일, 14일, 30일 연속 인증 배지)
- **목적**: 일일 성취감 제공 및 지속적 참여 동기 부여

#### 5.2.2 All Clear 보상
- **조건**: 그룹 기간 동안 100% 인증 완료
- **보상 내용**: 
  - 특별 포인트 30점 (일반 인증의 3배)
  - All Clear 달성 배지
  - 골드 테두리 및 반짝임 효과 (지나온 길 표시)
  - All Clear 인증서 (이미지 저장 가능)
- **표시**: "지나온 길"에서 특별하게 표시 (골드 테두리, 반짝임 효과)
- **목적**: 완주에 대한 성취감 및 보상 제공

#### 5.2.3 보상 사용처
- **포인트**: 
  - 식물키우기 기능에서 아이템 구매 (향후 확장)
  - 프로필 꾸미기 아이템 (향후 확장)
  - 기타 인게임 아이템 (향후 확장)
- **배지**: 프로필에 표시되어 성취 기록으로 활용
- **레벨**: 사용자 레벨 상승으로 장기적 동기 부여

### 5.3 보상 시스템의 목적

- **즉각적 피드백**: 매일의 작은 성취를 인정받음
- **장기적 동기**: All Clear를 향한 목표 설정
- **시각적 성취**: "지나온 길"을 통한 성취 시각화
- **지속적 참여**: 단계별 보상을 통한 지속적 참여 유도

- **유해 콘텐츠**: 성적, 폭력적, 혐오적 콘텐츠 업로드

- 위반 행위에 따라 경고, 게시물 삭제, 활동 정지, 계정 정지 등의 조치

#### 7.3.1 서비스 이용

#### 7.3.2 회원의 의무
