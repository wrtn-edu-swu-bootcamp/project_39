## 1. 디자인 철학

### 1.1 핵심 가치

**편안함 (Comfort)**
- 부드럽고 따뜻한 시각적 언어
- 넓은 여백과 호흡감 있는 레이아웃
- 부드러운 곡선과 둥근 모서리
- 눈에 피로를 주지 않는 색상 조합

**세련됨 (Sophistication)**
- 미니멀하고 정제된 디자인
- 일관된 디자인 시스템
- 세밀한 디테일과 완성도
- 현대적이고 트렌디한 감각

### 1.2 디자인 원칙

1. **접근성 우선**: 모든 사용자가 쉽게 이해하고 사용할 수 있는 인터페이스
2. **일관성**: 모든 화면과 컴포넌트에서 일관된 디자인 언어 사용
3. **계층 구조**: 명확한 시각적 계층으로 정보의 중요도 전달
4. **감성적 연결**: 사용자의 감정에 공감하고 지지하는 디자인
5. **기능성**: 아름다움과 실용성의 균형

## 2. 색상 시스템

### 2.1 색상 철학

원색을 피하고, 편안하고 세련된 무채색 기반의 색상 팔레트를 사용합니다. 부드러운 파스텔 톤과 차분한 톤온톤 색상을 조화롭게 배치하여 시각적 편안함을 제공합니다.

### 2.2 메인 컬러 팔레트

#### Primary Colors (주요 색상)

**Primary Blue (주요 블루)**
- `#6B8FA8` - 메인 액션 버튼, 강조 요소
- `#8BA8B8` - 호버/활성 상태
- `#5A7A8F` - 다크 모드 대응

**Primary Green (주요 그린)**
- `#7FA88B` - 성공, 완료 상태
- `#9BB8A5` - 연한 그린 (배경, 보조 요소)
- `#6B8F7A` - 다크 그린

#### Secondary Colors (보조 색상)

**Warm Gray (따뜻한 그레이)**
- `#A8A8A0` - 텍스트 보조 색상
- `#C8C8C0` - 배경, 구분선
- `#E8E8E0` - 연한 배경

**Soft Beige (부드러운 베이지)**
- `#D4C5B8` - 카드 배경, 섹션 구분
- `#E8DDD0` - 연한 배경
- `#C0B5A8` - 테두리, 구분선

**Muted Purple (차분한 퍼플)**
- `#9B8FA8` - 특별 기능, 강조
- `#B8A8C8` - 연한 퍼플 (배경)
- `#7F6B8F` - 다크 퍼플

### 2.3 상태 색상

**Info (정보)**
- `#8FA8B8` - 정보, 안내
- `#E8EBF5` - 정보 배경

### 2.4 중립 색상 (Neutral Colors)

**Gray Scale**
- `#1A1A1A` - Primary Text (다크 모드: `#FFFFFF`)
- `#4A4A4A` - Secondary Text (다크 모드: `#CCCCCC`)
- `#8A8A8A` - Tertiary Text (다크 모드: `#999999`)

## 4. 레이아웃 및 간격

### 4.1 그리드 시스템

**모바일 (기본)**
- Screen Width: `375px` (기준), `320px` ~ `428px` (범위)
- Content Padding: `16px` (좌우)
- Column Count: `1` (기본), `2` (그리드 레이아웃)
- Gutter: `12px` (컬럼 간 간격)

**태블릿**
- Screen Width: `768px` 이상
- Content Padding: `24px` (좌우)
- Column Count: `2-3` (그리드 레이아웃)
- Gutter: `16px` (컬럼 간 간격)

### 4.2 간격 시스템 (Spacing Scale)

간격은 `4px` 단위로 구성하여 일관성을 유지합니다.

- `4px` - 매우 작은 간격 (아이콘과 텍스트 사이)
- `8px` - 작은 간격 (관련 요소 간)
- `12px` - 기본 간격 (카드 내부 요소)
- `16px` - 중간 간격 (섹션 내 요소)
- `24px` - 큰 간격 (섹션 간)
- `32px` - 매우 큰 간격 (주요 섹션 간)
- `48px` - 최대 간격 (페이지 섹션 간)

### 4.3 여백 (Margin & Padding)

**카드 내부 패딩**
- `16px` - 기본 카드
- `20px` - 큰 카드
- `12px` - 작은 카드

**섹션 여백**
- `24px` - 섹션 상하 여백
- `32px` - 주요 섹션 간 여백
- `16px` - 관련 요소 간 여백

### 4.4 레이아웃 원칙

1. **호흡감**: 충분한 여백으로 시각적 편안함 제공
2. **균형**: 좌우 대칭과 비대칭의 조화
3. **계층**: 중요도에 따른 시각적 계층 구조
4. **일관성**: 모든 화면에서 동일한 레이아웃 규칙 적용

## 6. 아이콘 및 일러스트레이션

### 6.1 아이콘 스타일

**일관된 스타일**
- **선 스타일 (Outline)**: 기본 아이콘
- **채움 스타일 (Filled)**: 활성 상태
- **두께**: `1.5px` (선), `2px` (강조)
- **크기**: `16px`, `20px`, `24px`, `32px` (표준 크기)
- **색상**: 상태에 따라 변경 (활성: `#6B8FA8`, 비활성: `#8A8A8A`)

**아이콘 라이브러리**
- Material Icons (Outlined)
- Feather Icons
- 커스텀 아이콘 (필요 시)

### 6.2 일러스트레이션 스타일

**일러스트레이션 원칙**
- **부드러운 선**: 둥글고 부드러운 곡선

**사용 위치**
- 온보딩 화면
- 빈 상태 (Empty State)
- 에러 화면
- 성취 화면

- 댓글, 반응에 자연스럽게 사용

**사용 가이드**

- 급격한 움직임 지양
- 60fps 유지

**의미 있는 애니메이션 (Meaningful)**
- 사용자 행동에 대한 명확한 피드백
- 상태 변화의 시각적 표현
- 불필요한 애니메이션 지양

### 7.2 애니메이션 타이밍

**Duration (지속 시간)**
- **빠른 애니메이션**: `150ms` - 버튼 클릭, 토글
- **일반 애니메이션**: `250ms` - 카드 전환, 모달
- **느린 애니메이션**: `400ms` - 페이지 전환, 페이드

**Easing (이징 함수)**
- **Ease Out**: `cubic-bezier(0.4, 0, 0.2, 1)` - 기본
- **Ease In Out**: `cubic-bezier(0.4, 0, 0.2, 1)` - 부드러운 전환
- **Spring**: `cubic-bezier(0.68, -0.55, 0.265, 1.55)` - 탄성 효과 (선택적)

### 7.3 주요 인터랙션

#### 버튼 클릭
- **Scale**: `0.98` (클릭 시)
- **Duration**: `150ms`
- **Easing**: `ease-out`

#### 카드 호버/터치
- **Shadow 증가**: `0 4px 12px rgba(0, 0, 0, 0.08)`
- **Transform**: `translateY(-2px)` (선택적)
- **Duration**: `200ms`

#### 페이지 전환
- **Fade**: `opacity 0 → 1`
- **Slide**: `translateX(20px) → 0`
- **Duration**: `300ms`

#### 로딩 애니메이션
- **Skeleton Screen**: 콘텐츠 영역을 회색 블록으로 표시
- **Spinner**: 부드러운 회전 애니메이션
- **Progress Bar**: 진행 상태 표시

### 7.4 특수 애니메이션

#### All Clear 효과
- **반짝임**: `opacity 0.8 → 1` (2초 주기)
- **골드 테두리**: 미세한 글로우 효과
- **Duration**: `2000ms` (반복)

#### 식물 성장
- **Scale**: `0.9 → 1` (성장 시)
- **Duration**: `500ms`
- **Easing**: `ease-out`

## 9. 화면별 디자인 가이드

### 9.1 인증 및 온보딩

**로그인/회원가입 화면**
- 넓은 여백과 중앙 정렬
- 부드러운 그라데이션 배경 (선택적)
- 소셜 로그인 버튼: 각 플랫폼 브랜드 색상 사용
- 이메일 입력: 깔끔한 입력 필드

**온보딩 화면**
- 일러스트레이션 중심 레이아웃
- 페이지 인디케이터: 부드러운 원형 도트
- "건너뛰기" 링크: 오른쪽 상단, 작은 텍스트

### 9.2 메인 네비게이션

**하단 탭 바**
- 반투명 배경 (선택적)
- 활성 탭: Primary 색상으로 강조
- 배지: 작은 원형, 빨간색 (`#B88F8F`)

### 9.3 그룹 관련 화면

**그룹 검색 화면**
- 검색 바: 둥근 모서리, 아이콘 포함
- 필터/정렬 버튼: Secondary 스타일
- 그룹 카드: 2열 그리드, 이미지 + 정보

**그룹 상세 화면**
- 헤더 이미지: 전체 너비, 약간의 그라데이션 오버레이
- 정보 카드: 섹션별로 구분
- 참여 버튼: 하단 고정 (선택적)

**그룹 내 인증 게시물 (패들렛)**
- 3열 그리드 레이아웃
- 카드: 정사각형, 이미지 중심
- 무한 스크롤: 부드러운 로딩

### 9.4 커뮤니티 화면

**취미공유방**
- 카드형 게시글 리스트
- 프로필 이미지: 작은 원형
- 반응 아이콘: 부드러운 색상

### 9.5 개인 기록 화면

**지나온 길 첫 페이지**
- 프로필 중심 레이아웃
- 통계 카드: 2×2 그리드
- 부드러운 배경 그라데이션

**All Clear 그룹 카드**
- 골드 테두리와 배경
- 미세한 반짝임 효과
- 특별 배지 표시

**낮은 참여도 그룹 카드**
- 낮은 채도 색상
- 약간의 투명도 적용
- 시각적 차별화

### 9.6 프로필 및 설정

**프로필 화면**
- 프로필 이미지: 큰 원형, 중앙
- 통계 카드: 2×2 그리드
- 배지 컬렉션: 작은 그리드

**설정 화면**
- 리스트 형태, 섹션별 구분
- 토글 스위치: Primary 색상
- 구분선: 연한 회색
